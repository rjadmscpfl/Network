# Network Backup Automation

Python + Netmiko 기반 Cisco 장비 백업 자동화 스크립트

## Features

- Cisco ASA / IOS-XE / IOS 백업
- Multithread backup
- Config export
- Logging
- Summary report

## Requirements

- Python 3.10+
- Netmiko
- pandas
- python-dotenv
